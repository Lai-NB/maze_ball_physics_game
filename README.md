# \# Maze Ball Physics Game



A 3D physics-based maze game created using Processing.



##### \## Features

\- Physics-based rolling ball

\- Collision detection

\- Friction and restitution

\- Main menu system

\- Tutorial screen

\- High scores saving

\- 3D maze gameplay



##### \## Controls

\- W/A/S/D → Tilt board

\- Q/E → Adjust restitution

\- Z/C → Adjust friction



##### \## Built With

\- Processing (Java Mode)

\- P3D Renderer
